Error Handling Strategy

Error Type Source Recovery User Sees
Network Timeout APIClient Retry x3, exp. backoff Spinner in status bar
HTTP 401 / 403 APIClient Attempt token refresh → retry "Re-authenticating..."
Auth expired (fatal) AuthModule Clear token, show login Login prompt screen
Stream fetch failed Player Skip to next track "Skipping unplayable track"
mpv crash Player Restart mpv, resume queue Brief "Reconnecting..."
Rate limit (429) APIClient Wait & retry "Rate limited, retrying..."
